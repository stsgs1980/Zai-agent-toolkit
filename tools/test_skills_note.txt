Для дизайна и стилизации я использую несколько скиллов в зависимости от задачи:

Скилл
Когда использую
fullstack-dev	Полный стек — Next.js 16 + Tailwind CSS 4 + shadcn/ui. Компоненты, страницы, API-роуты, Prisma — всё вместе.
frontend-styling-expert	Когда нужна чисто визуальня работа: CSS, анимации, responsive, layout, hover-эффекты, цветовые схемы, центрирование, адаптив.
anti-monolith	Архитектура компонентов — когда файл разрастается, нужно разбить на модули, вынести хуки, сделать barrel exports.

В нашем Prompt Studio я в основном использую fullstack-dev — потому что это полный стек-проект (Next.js + API + UI), а для финишной полировки UI/анимаций могу подключить frontend-styling-expert.

Да, есть ещё! Вот все скиллы, которые могут быть полезны для визуальной/дизайнерской работы, которые я не упомянул:

Скилл
Для чего
charts	Профессиональные графики, диаграммы, дашборды, флоучарты, миндмапы, архитектурные диаграммы. ECharts, D3.js, Mermaid, Playwright+CSS.
image-generation	Генерация картинок через AI — иконки, иллюстрации, арт, визуальные ассеты для проекта.
web-shader-extractor	Вытягивает WebGL/Shader эффекты с сайтов — частицы, анимации фона, canvas-эффекты. Если увидел крутой эффект на сайте — можно вытащить и встроить.

Для Prompt Studio прямо сейчас могли бы быть полезны:
charts — сделать визуализацию оценки (score) как красивый gauge/radial chart вместо простого числа
image-generation — сгенерировать лого или иконки для интерфейса
web-shader-extractor — добавить живой анимированный фон (частицы/потоки)
